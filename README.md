# 📌 About The Project

**UNSOED Profile** adalah project website sederhana yang dibuat untuk menampilkan informasi profil mahasiswa seperti:

- Data diri
- Foto profil
- Informasi pendidikan
- Kontak atau media sosial

Project ini dibuat sebagai latihan pengembangan website menggunakan **HTML, CSS, dan JavaScript**.

---

# 🚀 Features

Beberapa fitur yang tersedia dalam project ini:

- Tampilan profil mahasiswa
- Informasi data diri
- Tampilan sederhana dan mudah dibaca
- Layout halaman yang rapi
- Bisa dijadikan template profil pribadi
